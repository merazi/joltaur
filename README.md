# Joltaur

A bash script to search and download packages from the AUR.

# Features

## Searching

You can search AUR packages by using the following command:

```sh
joltaur search <package>
```

## Downloading source code

You can download the source code of your preferred package by executing the following command:

```sh
joltaur download <package>
```

This will always default to downloading the latest version of the package.

## Flags

You can use the following flags to get more information about the script:

- `-h` or `--help`: Show the help message.
- `-v` or `--version`: Show the version of the script.
- `-s` or `--script`: Run the script with no colors.
